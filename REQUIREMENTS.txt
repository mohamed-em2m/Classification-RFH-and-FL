🛠️ Installation
git clone https://github.com/lucas-lacerda-de-souza/Classification-RFH-and-FL.git
cd Classification-RFH-and-FL
pip install -r requirements.txt
⚠️ Note: CUDA 12.8 and compatible NVIDIA drivers must be installed prior to PyTorch installation for GPU acceleration.
